== Installation ==

1. Upload the extracted `sample-plugin` folder to the `/my-plugins/` directory
2. Activate the plugin through the 'Plugins' menu in bbPress
3. Optional - Change the plugin's settings by going to `Settings` -> `Sample Plugin`
4. Enjoy!
